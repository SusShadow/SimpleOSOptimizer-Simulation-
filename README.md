# 🛠️ SimpleOSOptimizer (Simulation)

A C++ terminal-based OS optimization tool that runs scripts, shows system tasks, cleans temp files (simulation), and logs actions. Built for learning C++ console applications with file I/O and system calls.

✨ Features

Interactive terminal menu

Run scripts (batch files) from C++

Log user actions to a file (logs/log.txt)

Clean exit from the program

Looping menu for multiple operations

🎯 Learning Goals

Practice C++ file I/O and stream handling

Work with system commands using system()

Understand loops and menu-driven applications

Learn variable scoping inside switch cases

Introduce basic logging functionality

🚀 Future Improvements

Add timestamps to logs

Include more OS optimization scripts

Refactor repeated code into functions

Input validation to prevent invalid entries

Expand the menu for additional tools
